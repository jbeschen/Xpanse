# Xpanse: Solar System Economic Simulation

A single-player space economic simulation where you guide humanity's expansion from Earth across the solar system.

## Features
- Real solar system with simplified orbital mechanics
- Tiered resource system with production chains
- Dynamic market economy with supply/demand pricing
- AI factions competing for resources and trade
- Build outposts, colonies, and factories

## Requirements
- Python 3.11+
- Pygame 2.x

## Installation
```bash
pip install -r requirements.txt
```

## Running
```bash
python -m src.main
```

## Controls
- **Mouse Wheel**: Zoom in/out
- **Click + Drag**: Pan camera
- **Left Click**: Select objects
- **ESC**: Deselect / Open menu
